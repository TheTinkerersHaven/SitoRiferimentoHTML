Liste HTML

Le liste HTML sono di tre tipi: lista non ordinata, lista ordinata e lista con descrizioni.

Lista non ordinata

Si realizza con il tag <ul>

  • Elemento 1
  • Elemento 2
  • Elemento 3

Lista ordinata

Si realizza con il tag <ol>

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Lista con descrizioni

Si realizza con il tag <dl>

Elemento 1
- il primo elemento
Elemento 2
- il secondo elemento
Elemento 3
- il terzo elemento
