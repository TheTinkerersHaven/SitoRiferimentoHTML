Lato Client (Frontend)

È quello che viene visualizzato nel browser

Si realizza attraverso:

  • HTML (HyperText Markup Language)
  • CSS (Cascading Style Sheets)
  • JS (JavaScript)

A cosa servono?

  • HTML definisce la struttura della pagina web (testo, immagini, link...);
  • CSS definisce lo stile della pagina web ed è collegato ai file HTML;
  • JavaScript permette l'esecuzione di codice nella pagina web, "animandola".
