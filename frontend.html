Lato Client (Frontend)

È ciò che viene visualizzato nel browser.

Si realizza attraverso:

Linguaggio Utilizzo Logo
HTML (HyperText Markup Language) Definisce la struttura della pagina web (testo, immagini, link...).
CSS (Cascading Style Sheets) Definisce lo stile della pagina web ed è collegato ai file HTML.
JS (JavaScript) Permette l'esecuzione di codice nella pagina web, "animandola".
