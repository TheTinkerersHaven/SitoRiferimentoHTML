Lato Server (Backend)

È ciò che sta nel server e permette il funzionamento del sito web

Si realizza attraverso:

  • Google Apps Script
  • Java (Servlet/JSP)
  • VBScript
  • PHP
  • Python
  • etc...

Quale scegliere?

Dipende dalla funzione del sito che stai creando.

  • Google Apps Script è utile se hai bisogno di integrazione con le app di Google;
  • Java è un'ottima scelta per le applicazioni enterprise e dispone di una grande comunità e di un gran numero di framework;
  • VBScript è utile per i contesti legacy, specialmente in ambienti Windows, e non è consigliato per nuovi progetti;
  • PHP è facile da imparare ed ha una grande comunità;
  • Python è versatile e facile da imparare e dispone di molte librerie e molti framework.
